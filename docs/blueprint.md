# **App Name**: Portfolio Pro

## Core Features:

- Animated Card Navigation: Sleek, minimalist home screen layout with animated cards for each section (About, Projects, Research, Skills, Contact).
- Interactive Project Showcase: Interactive project previews with animated transitions.
- Contact Form: Contact form that validates input and provides feedback to the user, and then sends form details to Bernard.
- Responsive Design: Full responsiveness across mobile devices.

## Style Guidelines:

- Primary color: Deep indigo (#4B0082) for a sophisticated and professional feel.
- Background color: Light lavender (#E6E6FA) to provide a gentle, calming backdrop.
- Accent color: Soft amethyst (#9966CC) to highlight interactive elements.
- Body font: 'PT Sans' (sans-serif) for readability.
- Headline font: 'Space Grotesk' (sans-serif) for titles.
- Use minimalist icons to represent different sections and skills.
- Subtle transitions and depth effects to enhance user experience without being distracting.
